# Rusty Pong

A small Pong clone written in Rust with `macroquad`.

## Controls

- `W` / `S`: move the left paddle
- `Up` / `Down`: move the right paddle
- `Space`: restart after a win

## Run

```bash
cargo run
```

This project was AI-built in about 20 seconds by GPT-5.4.
